# ①課題名
TriArk お気に入りホスト管理アプリ（ブックマークアプリ その2）

## ②課題内容（どんな作品か）
気になるホストや体験をブックマークして管理できるCRUDアプリです。
登録・一覧表示に加えて、今回は前回講義で学習した編集（更新）機能と削除機能を追加しました。
PHPとMySQL（PDO）で構築し、旅の体験を提供するサービス「TriArk」の
お気に入り管理機能を想定しています。

主な機能：
- 登録（insert.php）
- 一覧表示（select.php）
- 編集・更新（detail.php → update.php）
- 削除（delete.php、削除前に確認ダイアログを表示）

## ⑤工夫した点・こだわった点
- 削除ボタンを押した際に「本当に削除してもよいですか？」の確認ダイアログを表示し、
  誤操作を防ぐようにしました。
- 共通処理（DB接続・エラー表示・ページ遷移・XSS対策）を 講義で学習したようにfuncs.php にまとめ、
  各ファイルから呼び出す構成にして、コードの重複を減らしました。
- セキュリティ対策として、表示時は htmlspecialchars でXSS対策、
  DB操作時はプレースホルダ（bindValue）でSQLインジェクション対策を行いました。
- ローカル（XAMPP）と本番（さくらインターネット）の接続情報を db_config.php 内で
  コメントアウトにより切り替えられるようにし、db_config.php は .gitignore で
  保護してGitHubに公開しないようにしました。
- 絵文字も保存できるよう、文字コードを utf8mb4 に対応させました。
