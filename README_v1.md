# ①課題名
TriArk お気に入りホスト ブックマーク機能（DB版）

## ②課題内容（どんな作品か）
- 旅プラットフォーム「TriArk」で、気になるホスト・体験をブックマークできる機能
- PHP + MySQL で「登録（index.php）→ DB保存（insert.php）→ 一覧表示（select.php）」を実装
- ホスト名・URL・メモを入力し、MySQLデータベースに保存。登録日時は自動記録
- 前回CSVで作った予約システムを、今回はデータベース版として発展させた

## ⑤工夫した点・こだわった点
- データの保存先をCSVからMySQLデータベースへ移行し、PDO + プレースホルダで安全にデータを扱う実装に
- 前課題同様ブランドカラー（ネイビー #1A2E3A / シアン #00B4C8 / オレンジ #FF8C42）でUIを統一
- DB接続情報を db_config.php に分離し、.gitignore で除外してGitHubに公開しない設計に
- データが0件のときは「まだありません」と表示する空状態に

## ⑥難しかった点・次回トライしたいこと（又は機能）
- ローカル（XAMPP）とさくら本番でDB接続情報が異なり、localhost のままでエラー（No such file or directory）に。　 db_config.php で切り替える設計にした

- 次回（Day3）で学ぶ編集（UPDATE）・削除（DELETE）を追加し、CRUDを完成させたい
